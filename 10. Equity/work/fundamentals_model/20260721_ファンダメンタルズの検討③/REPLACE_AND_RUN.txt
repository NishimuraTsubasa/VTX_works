個別銘柄スコアリングモデル v0.12.2 全差し替え・実行手順

1. 旧プロジェクトを別フォルダへバックアップしてください。
2. ZIPを展開し、stock_scoring_model_v12_2 フォルダをVS Codeで開いてください。
3. 実データを data/input/factors_and_returns.xlsx へ配置してください。
4. data/input/factor_master.xlsx の設定を確認してください。

重要な初期設定
- 入力履歴開始: 2018年想定
- Layer1最低学習: 18か月
- Layer1モデル選択: 6か月
- Layer3最低学習: Layer1 OOF開始後12か月
- Layer3 lookback: 36か月
- Ridge alpha検証: 6か月
- S07_OLS_Linear: 有効
- S07_Ridge_Linear: 有効
- S07_Ridge_Flexible: 無効

PowerShell実行順

py -m pip install -e .
py scripts\validate_inputs.py --config config\model_config.py
py scripts\run_pipeline.py --config config\model_config.py

テスト

py -m pytest -q

VS Codeデバッグ
- .vscode/launch.json
- Debug scoring pipeline
- エントリーポイント: scripts/run_pipeline.py

主比較
- outputs/analysis_summary.xlsx の共通OOS指標
- outputs/stock_scoring_scenario_comparison.pdf
- outputs/s07_ols_ridge_comparison.xlsx
- outputs/s07_ols_ridge_comparison.pdf

注意
- 各モデルの全利用可能期間のMean RankICは評価期間が異なるため、主判断には使わないでください。
- 主判断はCommonMeanRankIC等の共通OOS指標を使用してください。
- 同梱サンプルは合成データです。
