# 分析方法

## 1. 目的

現行の0-1順位単純平均から処理を一つずつ追加し、個別銘柄の翌期リターン順位付け精度がどの段階で改善するかを確認します。

## 2. 共通評価

各時点でTotalScoreを5分位に分けます。

- Q1: 最低20%
- Q5: 最高20%

各分位の翌期等ウェイトリターンとQ5-Q1を計算します。同時に、TotalScoreと翌期リターンのSpearman RankICを計算します。

## 3. 望ましい結果

- Q1 < Q2 < Q3 < Q4 < Q5 が概ね成立
- Q5-Q1累積リターンが安定上昇
- 平均RankICが正
- RankIC正符号率が50%を上回る
- 特定月だけに依存しない
- 複雑モデルの改善がOOSでも維持される

## 4. 公平な比較

S00-S07は、すべてのパターンで予測値と翌期リターンが存在する共通期間に限定して比較します。これにより、長い履歴を持つ単純モデルが有利になることを防ぎます。

## 5. モデル判定

主指標はMean RankICとQ5-Q1 Sharpeです。補助として単調性、最大ドローダウン、正符号率を確認します。複雑モデルの改善が標準誤差内であれば、単純モデルを優先します。

## 派生ファクターの追加

設定されたグループ・FAコードについて、原系列のほかに差分、移動平均乖離、移動平均比率、過去平均乖離を生成できます。派生特徴量は独立したファクターコードとして前処理・ビン分析・4候補モデル比較へ投入します。

スコア時点tに格納する標準式は次の通りです。

```text
DIFF_P1_L1   = x[t-1] - x[t-2]
MADEV_W12_L1 = x[t-1] - mean(x[t-2], ..., x[t-13])
```

評価対象はt+1リターンであるため、最新使用情報t-1との間隔は2期です。
